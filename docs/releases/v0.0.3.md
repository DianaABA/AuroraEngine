# AuroraEngine v0.0.3

Release date: 2025-11-19

Highlights:
- Utils: new `Gallery` module for CG unlocks with persistence and `gallery:unlock` events.
- Template: Gallery panel UI with grid; Expressions demo unlocks a sample CG via a `flag`.
- Engine: runtime toggles `setAutoAdvance`/`setAutoDecide` and `isAutoAdvance`/`isAutoDecide` for Auto/Skip friendliness.
- Template: Auto and Auto-Choose buttons wired to runtime toggles.

Breaking changes:
- None.

Docs:
- New: `docs/gallery.md`.
- Template README updated to mention Expressions demo, Gallery, and Auto controls.
- Changelog updated.

Try it:
- From repo root, run `npm run template:minimal:dev`, click "Start Expressions", advance to unlock the CG, and open the Gallery.
