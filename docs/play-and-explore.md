# Play & Explore (No Coding)

Quick steps to experience AuroraEngine.

1. Open the live demo: https://chakrahearts.netlify.app/ (or run `npm run dev` in `templates/minimal` if local).
2. Pick a pack in the dropdown:
   - Example — branching basics
   - Expressions — sprites + CG unlock
   - Achievements — badge unlock path
   - RTL/TextId — Arabic + textId table
   - BYOK — API key demo (optional)
3. Controls:
   - Click/tap or press Space/Enter to advance.
   - Click choices to branch.
   - Quick Save/Load buttons store/restore progress.
4. Gallery/Achievements/Codex buttons show unlocked items as you play.
5. To try your own story:
   - Scroll to “Load Story File”.
   - Paste a story JSON or pick a file, then click “Check Story” and “Start Story”.

That’s it—no setup or coding required.***
